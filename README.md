# Content Marketing Hub

Central content operations repo for all Dennis Rankin projects.

## Purpose
This repo is the single source of truth for:
- Daily social media posts
- Weekly campaign plans
- Platform-specific content calendars
- Brand assets/logos used in marketing content

## Projects Covered (initial)
- Addressed
- CipherLabs

## Folder Structure
- `assets/logos/` → reusable logo files by project
- `calendar/` → posting schedules (daily + weekly)
- `campaigns/` → campaign briefs and execution plans
- `posts/` → drafted copy by platform/date
- `docs/` → process, SOPs, and publishing workflow

## Operating Cadence
- **Daily:** publish-ready post drafts for active channels
- **Weekly:** campaign theme, objective, hooks, and CTA plan
- **Monthly:** content performance review + strategy update

## Ownership
Managed by Amara (content marketing assistant) with Dennis approval before publishing.
