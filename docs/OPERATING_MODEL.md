# Operating Model: Social Media Management

## Mission
Run consistent, high-quality social content across Dennis's projects to drive awareness, trust, and conversions.

## Workflow
1. Weekly strategy brief (audience, offer, angle)
2. Daily post generation (platform-specific copy + visual prompt)
3. Review/approval pass (Dennis)
4. Scheduling/publishing
5. Lightweight performance notes

## Cadence
- **Daily posts:** 1-2 per active platform
- **Weekly campaigns:** one focused campaign with 3-7 coordinated posts
- **Weekly review:** what worked, what to iterate

## Content Types
- Education (tips, insights)
- Product-led (feature/value)
- Story-led (founder POV, behind the scenes)
- Conversion (CTA to sign up, demo, or waitlist)

## Guardrails
- Drafts only until approved
- No unverifiable claims
- Consistent voice per project
